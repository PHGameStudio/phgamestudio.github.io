# Markdown 语法速查指南

欢迎使用 Markdown！这是一种轻量级的标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后将其转换成有效的 XHTML（或者 HTML）。

本文档旨在介绍最常用的 Markdown 语法。

---

## 1. 标题 (Headers)

使用 `#` 号可表示 1-6 级标题，一级标题对应 HTML 的 `<h1>` 标签。

**语法：**
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

---

## 2. 文本样式 (Text Styling)

常用的文本强调方式。

| 效果 | 语法 | 示例 |
| :--- | :--- | :--- |
| **加粗** | `**文本**` 或 `__文本__` | **Bold** |
| *斜体* | `*文本*` 或 `_文本_` | *Italic* |
| ***粗斜体*** | `***文本***` | ***BoldItalic*** |
| ~~删除线~~ | `~~文本~~` | ~~Deleted~~ |

---

## 3. 列表 (Lists)

### 无序列表
使用 `-`、`+` 或 `*` 标记。

**语法：**
```markdown
- 第一项
- 第二项
  - 嵌套项
  - 嵌套项
- 第三项
```

**效果：**
- 第一项
- 第二项
  - 嵌套项
  - 嵌套项
- 第三项

### 有序列表
使用数字加点 `1.` 标记。

**语法：**
```markdown
1. 第一步
2. 第二步
3. 第三步
```

**效果：**
1. 第一步
2. 第二步
3. 第三步

---

## 4. 链接与图片 (Links & Images)

链接和图片的语法非常相似，区别在于图片前面多一个感叹号 `!`。

### 链接
**语法：** `[链接文字](链接地址 "可选标题")`
**示例：** `[Google](https://www.google.com)`
**效果：** [Google](https://www.google.com)

### 图片
**语法：** `![Alt文本](图片地址)`
**示例：** `![Markdown Logo](https://markdown-here.com/img/icon256.png)`

---

## 5. 引用 (Blockquotes)

使用 `>` 来引用一段文字。

**语法：**
```markdown
> 这是一个引用段落。
>
> > 这是一个嵌套引用。
```

**效果：**
> 这是一个引用段落。
>
> > 这是一个嵌套引用。

---

## 6. 代码 (Code)

### 行内代码
使用反引号 \` 包裹代码。

**语法：** `使用 `git status` 检查状态`
**效果：** 使用 `git status` 检查状态

### 代码块
使用三个反引号 \`\`\` 包裹，并指定语言（可选）以实现语法高亮。

**语法：**
\```python
def hello():
    print("Hello, World!")
\```

**效果：**
```python
def hello():
    print("Hello, World!")
```

---

## 7. 表格 (Tables)

使用 `|` 分隔单元格，使用 `-` 分隔表头。
可以使用 `:` 控制对齐方式（左对齐、右对齐、居中）。

**语法：**
```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :----: | -----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```

**效果：**

| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :----: | -----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

---

## 8. 任务列表 (Task Lists)

适用于待办事项。

**语法：**
```markdown
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 待办事项
```

**效果：**
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 待办事项

---

## 10. 特殊字符转义

如果你想显示 Markdown 中的特殊字符（如 `*` 或 `#`），请在前面使用反斜杠 `\`。

**示例：** `\# 不是标题` 显示为 \# 不是标题
